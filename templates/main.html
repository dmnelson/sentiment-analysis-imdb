Sentiment Analysis IMDB Reviews

Showcase of a project developed for the Natural Language Processing course at UFMG. This aims to classify movie reviews based on knowledge acquired by training a recurrent neural network (LSTM) on the IMDB reviews database.

The text field below can be used to test the algorithm by classifying sentences representing hypothetical movie reviews, for example "I liked the movie" or "That movie was really boring". Further below on the page there are some examples extracted from the dataset comparing the performance of the algorithm to the actual polarity.
